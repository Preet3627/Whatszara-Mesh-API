Welcome to Whatszara

Your desktop assistant controlled via WhatsApp + LLM.

  1. Start the WhatsApp bridge: cd whatsapp-bridge && go run main.go
  2. Start the orchestrator: cd orchestrator && python main.py
  3. Configure an LLM provider in Settings
  4. Send a message to your WhatsApp number to control your desktop
