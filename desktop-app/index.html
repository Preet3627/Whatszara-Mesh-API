Troubleshooting

              

                

                  Bridge stopped or unavailable
                  
Confirm Go is installed with go version, then restart the app.

                

                

                  QR code does not appear
                  
Remove whatsapp-bridge/store/, restart, and link the device again.

                

                

                  Models are missing
                  
For Ollama, start ollama serve and pull a model such as ollama pull llama3.1.

                

                

                  Messages do not execute actions
                  
Check the contact mode, allowlist entry, and tool permission toggles.
